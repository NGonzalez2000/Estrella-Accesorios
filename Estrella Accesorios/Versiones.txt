[FORMULARIOS]
* Frm_Articulos:
		- ABM (Codigo, Cod. Barras, Descripcion, Importe Compra, Importe Venta, Fecha Modificación, Stock Inicial, Stock Minimo, Imagen, Familia, Sub-Familia, Proveedor)
		- Modificacion por Lotes de Precios (Porcentaje, Suma, Resta, Multiplizacion, Division, Valor [+ y -])
		- Busqueda por Codigo, Codigo de Barras, Descripicon, Familia, Sub Familia, Proveedores

* Frm_Configuracion:
		- Conexion con Base de Datos y Backup
		- Dias de Alerta en Precios (Venta)
		- Puerto de Conexion Lector Barras / QR (RS232)
		- Configuración de colores para estado de STOCK

* Frm_Etiqueta:
		- ABM de Listado de Articulos y Cantidad de etiquetas para el soporte de estanteria y generacion de codigo QR

* Frm_Familias:
		- ABM de Familias y Sub-Familias

* Frm_Forma_Pago:
		- ABM de distintas formas de pago, con descuentos y recargos para las ventas

* Frm_Ingresos:
		- Altas de cantidades de articulos ingresados al stock, permite ademas modificar los datos de los articulos ya existentes.
		- Las cantidades ingresadas se permite cargar a las etiquetas de articulos.

* Frm_Login:
		- Valida el ingreso al sistema con un nombre de usuario y password, devolviendo si es administrador o no para habilitar distintas funciones.

* Frm_Principal:
		- Página principal de sistema, permite ingresar a:
				+ Articulos				-> Abre Frm_Articulos
				+ Ventas				-> Abre Frm_Ventas
				+ Ingresos				-> Abre Frm_Ingresos
				+ Estadisticas			-> Abre Frm_Estadisticas (aun no desarrollado)
				+ Opciones:
					+ Funciones Varias:
						+ Exportar Datos:
							- Permite exportar la base de datos y las imagenes de los articulos a una ubicacion elegida por el usuario
						+ Importar Datos
							- Permite importar datos de articulos, proveedores, familias, sub-familias, usuarios y las imagenes de los articulos desde una ubicacion elegida por el usuario
						+ Etiquetas QR					-> Abre Frm_Qr
						+ Etiquetas Para Estanterias	-> Abre Frm_Etiquetas
						+ Configuracion del sistema		-> Abre Frm_Opciones
					+ Usuario			-> Abre Frm_Usuarios
					+ Forma de Pagos	-> Abre Frm_Forma_Pago
					+ Proveedores		-> Abre Frm_Proveedores
					+ Familias			-> Abre Frm_Familias
				+ Salir del Sistema:
							- Cierra el aplicativo.

* Frm_Proveedores:
		- ABM de Proveedores

* Frm_QR:
		- ABM de Etiquetas de articulos con generacion de codigo QR

* Frm_Usuarios:
		- ABM de Usuarios (Nombre, Password, Administrador, Habilitado)

* Frm_Ventas:
		- Arma el listado de articulos vendidos, permitiendo ingresar codigo (o buscarlo) y cantidad, mostrando los datos mas significativos
		- Muestra si el articulo esta en stock pero permite venderlo si no fugura en sistema
		- Muestra advertencia de dias sin actualizar el precio
		- Calcula descuentos y recargos segun forma de pago elegida
		- Calcula vuelto

* Frm_Vista_Previa:
		- Muestra la vista previa de informes y etiquetas para su impresion o exportacion


[REPORTES]
* Rpt_Etiquetas: Etiquetas para articulos (QR, codigo, descripcion) - Medidas (3.56 x 1.69 - A4)
* Rpt_Etiquetas1: Etiquetas para articulos (QR, codigo) - Medidas (3.56 x 1.69 (2 por etiqueta) - A4)
* Rpt_Etiquetas_Estanteria: Etiquetas para estanterias (QR, codigo, descripcion, precio venta) - Medidas (5 x 3 - A4)
* Rpt_Ventas: Reporte detallado de venta realizada, Fecha, Cantidad, Descripcion, Importe Unitario, Total Linea, Total General) - Medidas (A4)
